MyFlix API

The backend of this project is a database in MongoDB, which is connected to a Node Application, hosted on Heroku. The front end will make calls to this API. There are methods for signing up a new user, logging in, and editing a user profile.

Besides methods for getting user information, the same API will also serve the JSON data about all the movies.
